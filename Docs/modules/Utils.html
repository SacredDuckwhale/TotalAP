Colours.lua

Utilities for dealing with and translating between different ways of representing colours
HexToRGB (hexString)
Translates HTML colour codes to RGB values

Parameters:

  • hexString A string representing a colour code in hexadecimal/HTML notation (the leading '#' symbol is optional)

Returns:

  1. Red value; 0 if invalid string was given
  2. Green value; 0 if invalid string was given
  3. Blue value; 0 if invalid string was given

Usage:

    HexToRGB("#FFFEFD") -> { 255, 254, 253 }
RGBToHex (r, g, b[, addPrefix])
Translates RGB values to HTML colour codes

Parameters:

  • r Red value
  • g Green value
  • b Blue value
  • addPrefix Whether or not to prefix the result with a '#' symbol (as used in HTML notation) (optional)

Returns:

    The hexadecimal/HTML representation of the colour

Usage:

  • RGBToHex(0, 255, 1) -> "00FF01"
  • RGBToHex(0, 255, 1, true) -> "#00FF01"

FQCN.lua

Utilities for dealing with fully-qualified character names
GetFQCN ([characterName[, realm]])
Build a fully-qualified character name from the given character and realm names (uses the current character if none are given)

Parameters:

  • characterName The name of a character (optional)
  • realm The realm said character exists on (optional)

Returns:

    Fully-qualified character name that can directly be used as a unique lookup key for many tables

Usage:

  • GetFQCN("Duckwhale", "Outland") -> "Duckwhale - Outland"
  • GetFQCN() -> "Duckwhale - Outland" (if this is the currently active character
  • GetFQCN("Swimmingly") -> "Swimmingly - Outland" (if the currently active character is located on the realm "Outland")
  • GetFQCN(nil, "Outland") -> "Swimmingly - Outland" (if "Swimmingly" is the name of the currently active character, regardless of the realm)

Format.lua

Utilities for number formatting and parsing
FormatShort (value[, format])
Format number as short textual representation Adheres to international standards for orders of magnitude ('k' for thousands, 'm' for millions, etc.)

Parameters:

  • value The number (value) to be formatted
  • format Whether or not the formatting should be applied directly. Will return the original value alongside a format string that can be used with string.format(...) otherwise (optional)

Returns:

  1. The formatted output if the 'format' parameter was given; a format string otherwise
  2. The number this formatted string can be applied to in order to obtain the same textual representation; nil if the 'format' parameter wasn't given

Usage:

  • FormatShort(15365, true) -> "15.3k"
  • FormatShort(15365) -> { "%.1fk", 15.365 }

TableFunctions.lua

Utilities for dealing with tables
CompareTables (t1, t2, targetTable, lastTableKey)
Compares two tables, and replaces mismatching entries in t2 with those from t1 (with t2 and t2 being used recursively, while targetTable remains the original table reference)

Parameters:

  • t1 The master table
  • t2 The table that is to be checked against the master table
  • targetTable Temporary table based on t2 that will have mismatching entries replaced by those from t1[lastTableKey]
  • lastTableKey The key last used before entering a new recursion
