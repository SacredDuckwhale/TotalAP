SlashCommands.lua

Handling of slash commands and console-based operations
GetSlashCommand ()
Returns the currently used (main) slash command

Returns:

    The slash command that is currently assigned to the addon
GetSlashCommandAlias ()
Returns shorthand (alias) for the currently used slash command

Returns:

    A shorter command that is also assigned to the addon, but may be overwritten by other addons more easily
GetSlashHandlerFunction (command)
Returns the handler function for a given slash command (if it exists)

Parameters:

  • command The slash command that is to be accessed

Returns:

    The function that should be called
PrintSlashCommands (usedAlias)
Prints a list of all available slash commands to the DEFAULT_CHAT_FRAME (using addon-specific print methods with colour-coding)

Parameters:

  • usedAlias Whether or not the actual (primary) slash command has been used
SlashCommandHandler (input, usedAlias)
Handles console input (slash commands). Called via AceConsole mainly TODO: Only one argument is supported currently (use AceConsole:GetArgs to parse them more easily)

Parameters:

  • input
  • usedAlias
SlashCommandHandler_UsedAlias (input)
Notify slash command handler that the alias slash command was used instead of the regular one before calling it

Parameters:

  • input
