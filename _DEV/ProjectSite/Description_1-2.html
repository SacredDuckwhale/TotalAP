Features

All of the main features can be disabled/toggled via keybindings and slash commands, allowing you to change the way most of the information is displayed.

Tooltip Enhancement

Adds tooltip information for artifact power items. Currently, the following can be displayed:

  • Number of artifact power tokens in the player's inventory
  • Total artifact power in bags
  • Progress towards next trait
  • Available number of traits (if applicable)

The tooltip can be customized somewhat, allowing users to decide which parts (if any) they want to include.

Action Button

TotalAP also displays a button whenever artifact power items are in the player's inventory. It can be used to quickly use the AP items without having to find them manually in your bags.

  • Indicates that new traits are available via spell overlay (glow effect similar to that of spell procs)
  • Includes keybinds (via Blizzard's KeybindUI) to use AP items, or hide the button manually
  • Button can be skinned using the Masque (formerly ButtonFacade) addon library
  • Hold ALT-Click while dragging to move the button, or SHIFT-Click while dragging to change its size

Bar Display

It is now possible to display the current progress of any spec's artifact weapon while considering all AP in the player's inventory as well.

  • Separate colours are used to indicate the amount of artifact power used on the weapon vs. AP items that still remaining unused
  • Each specialization's progress will be cached when activated, and frequently while it remains so
  • If a spec isn't displayed initially, simply switch to it once to have it be updated
  • Hold ALT-Click while dragging to move them, along with the rest of the display

Specialization Icons/Text

Another means of portraying the progress for all specs is in the form of these icons.

  • Indicates that new traits are available via similar spell overlay (glow effect) as the action button
  • Displays progress and number of available traits as text next to each icon
  • Clicking the icon for a particular specialization activates it
  • Also supports Masque

Slash commands

The following slash commands are provided: /totalap and /ap (alias)

A more detailed list will be printed ingame (see screenshots).
